# Crossword-Print-Dynamic-C-Lang-Mini-Project

* Contains C implementation of the Crossword Dynamic Printing.
* It creates a **crossword with _ acting as fillups generated randomly or dynamically on each run**. 


<hr/>
<b>Language:</b> <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/c/c-original.svg" height=50>


<b>Created (date) :</b> Sept 2019


<b>Created in :</b> F.Y. B.Sc (SEM I)


<b>Created for :</b> Mini Project of C Language Subject


<b>Logic Description:</b> Two for loops calling the words functions and the generate_across or generate_down function to generate the structure of the words in order with random _ to guess.
